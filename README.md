# Operation Radiohead Dossier

This repository contains the source code for the "Operation Radiohead" secure dossier interface. It is a React-based web application designed to present critical intelligence regarding a global neural hijacking crisis, along with the strategic vision and counter-measures provided by the Stellarium Foundation.

## Overview

The application features a single-page, tab-driven architecture with the following main sections:
- **Home:** Executive summary of the ongoing crisis.
- **Systems (Psychology & Controls):** Analysis of neural conditioning mechanisms and counter-technologies.
- **Stellarium:** The blueprint and vision of John Victor, outlining the "Water" suite of AI tools and the Enterprise Housing Pledge.
- **Crisis:** A registry of atrocities, war crimes, and victim intelligence.
- **Support:** Sovereign funding protocols, cryptocurrency (Monero) donation channels, and global sanctuary locations.
- **Contact:** An encrypted intelligence feed for reporting and coordination.
